# Home-Meteo-Station
A simple home meteo station based on Arduino Uno.

# Measuring parameters
- temperature (oC)
- humidity (%)
- atmospheric pressure (mBar)
- CO2 (ppm)

# Features
CO2 zero-reference calibration.
