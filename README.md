# Home-Meteo-Station
A simple home meteo station was build using Arduino Uno.

# Features
Measuring parameters:
- temperature (oC)
- humidity (%)
- atmospheric pressure (mBar)
- CO2 (ppm)
CO2 zero-reference calibration.
